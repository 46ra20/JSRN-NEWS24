Question#NO-1:- What is different Var,Let and Const?

Var:- It's not a scope function , you can use it's any ware any time.

Let:- Let is a method of declare variable. you can use this local and globally but can't call local variable as a global variable.

Const:- One time declaration. You can declare a variable with this key word but can't change or update this.

Question#NO-2:- What is different Arrow function and Regular function?

Arrow Function:- Nothing different between Arrow function and Regular function with out declaration method. ()=>{} arrow function

Regular Function:- function(){}

Question#NO-3:- What is map, find, filter and forEach function?

Map:- map() function call all element of array and manipulate this element.

Find:- find() function call an array and match element with logic and return first matching value.

Filter:- filter() function call all array element and match with logic return all matching element

forEach:- forEach() function call all array element. forEach() function not execute for empty elements.
